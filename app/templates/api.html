API

Scan any domain programmatically. Free, no signup, just curl.

1. Scan a Domain

GET /api/scan?domain=example.com

Scan a domain and get JSON with 11 security modules, A-F grade, and findings.

curl "https://contrastcyber.com/api/scan?domain=example.com"

2. Download Report

GET /api/report?domain=example.com

Scan + download a human-readable .txt security report.

curl "https://contrastcyber.com/api/report?domain=example.com" -o report.txt

3. Bulk Scan PRO

POST /api/v1/bulk

Scan up to 50 domains in a single request. Requires a Pro API key.

POST /api/v1/bulk HTTP/1.1
Host: contrastcyber.com
X-API-Key: <your-pro-key>
Content-Type: application/json

{"domains": ["example.com", "google.com"]}

Response

{
  "results": [
    {
      "domain": "example.com",
      "scan_id": "a1b2c3...",
      "grade": "A",
      "score": 92,
      "result": { ... }
    },
    {
      "domain": "google.com",
      "scan_id": "d4e5f6...",
      "grade": "D",
      "score": 55,
      "result": { ... }
    }
  ],
  "total": 2,
  "successful": 2
}
Header X-API-Key — your Pro API key
Body {"domains": [...]} — max 50 domains per request
Limit 1,000 scans/hour per key
Note Duplicate domains are automatically deduplicated. Failed domains don't block others.

Other Endpoints

GET /report/{scan_id}.txt — download report by scan ID
GET /badge/{domain}.svg — dynamic SVG grade badge for README files
GET /result/{scan_id} — web result page (HTML)
GET /stats — aggregate scan statistics

Rate Limits

Free 100 scans/hour per IP — no signup needed
Pro 1,000 scans/hour per key + bulk scan (up to 50 domains/request)
Per domain 10 scans/hour (all users combined)

Error Codes

400 Invalid domain format
401 Missing or invalid API key (bulk scan)
429 Rate limit exceeded
502 Scan failed (domain unreachable)
504 Scan timed out
