About ContrastScan

ContrastScan is a free, open-source website security scanner built by an independent developer. It checks 11 security aspects of any website and returns a single A-F grade out of 100 points in under 3 seconds. The scanner engine is written in C for speed. The web backend is Python (FastAPI). The entire codebase is open source on GitHub with 900+ automated tests.

ContrastScan is not affiliated with Contrast Security (contrast.com). They are completely different products.

We do not collect personal data, require accounts, or show ads. Scan results are stored temporarily for caching purposes only.
