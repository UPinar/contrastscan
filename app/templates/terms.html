1. Service Description

ContrastCyber ("we", "us") operates ContrastScan at contrastcyber.com — a free website security scanning tool that analyzes publicly accessible security configurations including SSL/TLS, HTTP security headers, DNS records (SPF/DKIM/DMARC), CORS, cookies, HTTPS redirect, DNSSEC, HTTP methods, information disclosure, HTML, and CSP. Scans return a 100-point score and A–F grade across 11 modules, plus background passive recon (WHOIS, DNS records, subdomains, certificate transparency logs, robots.txt, sitemap).

Our security intelligence API is a separate property at api.contrastcyber.com with its own Terms and Privacy Policy.

The Service is provided as-is with no warranty.
